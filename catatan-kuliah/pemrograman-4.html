Tipe Data

Seperti namanya sendiri, tipe data berarti tipe data. Tipe-tipe ini mengelompokkan data apa yang ditunjukkan dalam program. Hampir semua bahasa pemrograman mendukung tipe data dasar bilangan (dengan berbagai sistem), karakter (dan bentukannya, string), dan Boolean.


Di Python, tipe data dasar yang tersedia adalah str, int, float, complex, bool. Di JavaScript, tipe data dasar yang tersedia adalah string, symbol, number, bigint, boolean, undefined, dan null.


Tipe data bilangan

Tipe data bilangan hampir meliputi semua sistem bilangan. Bilangan bulat dan pecahan merupakan tipe data yang sering ditemui dalam pemrograman. Bilangan bulat (integer) terdiri dari bilangan negatif, nol (0), dan bilangan positif. Pecahan direpresentasikan sebagai floating-point. Operasi-operasi yang berlaku adalah operasi aritmetika dan operasi perbandingan.


Tipe data string

Tipe data string meliputi semua karakter yang mungkin (biasanya Unicode). Dalam beberapa bahasa pemrograman seperti C/C++, tipa data string merupakan tipe data bentukan dari array karakter, berbeda dengan Python dan JavaScript yang memiliki tipe data string. Namun, keduanya sama-sama bisa diakses karakternya dengan indeks layaknya array. Operasi-operasi yang berlaku adalah operasi perbandingan dan operasi penyambungan.


Tipe data Boolean

Nama "Boolean" berasal dari matematikawan asal Inggris yang pertama kali memperkenalkan konsep ini, George Boole (1815-1864). Tipe data ini hanya meliputi "Benar" dan "Salah", tetapi tidak bisa keduanya atau di antaranya. Operasi yang berlaku adalah operasi perbandingan.
