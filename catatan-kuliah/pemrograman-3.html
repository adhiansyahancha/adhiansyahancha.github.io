Kode Semu

Kode semu, terjemahan dari pseudocode, adalah deskripsi tulisan yang bentuknya menyerupai (pseudo-) kode program tetapi bukan kode program asli.


Dalam pengajaran di perguran tinggi, cara menulis kode semu dibagi menjadi tiga bagian, yaitu program, deklarasi, dan algoritma. Bagian program menunjukkan judul program. Bagian deklarasi berisi variabel yang dinyatakan. Bagian algoritma merupakan bagian utamanya.


Berikut di bawah ini merupakan contoh kode semu untuk menghitung nilai absolut.

program NilaiAbsolut
        
deklarasi
    a: integer
                
algoritma
    read(a)
    if a < 0 then
        write(-a)
    else
        write(a)
    end
