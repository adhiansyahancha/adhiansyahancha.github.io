Selamat datang di Catatan Kuliah Rekayasa Perangkat Lunak

Sebenarnya ide ini sudah ada ketika saya memasuki semester 1 di Institut Teknologi Telkom Purwokerto, tetapi beberapa kali terhenti karena pengembang situs ini sibuk dan malas untuk tetap aktif di GitHub Pages. Memasuki semester 2, penulis mulai merasakan bahwa mencatat di situs web pribadi lebih baik untuk mencatat materi ketimbang di kertas khusus.


Bukan berarti catatan ini dapat menggantikan catatan di kertas maupun pendidikan di kelas. Mencatat di kertas tetap penulis lakukan untuk keperluan yang khusus. Pengembangan catatan di situs ini juga dimotivasi oleh keinginan penulis untuk mengembangkan kemampuan mengkodenya di web (HTML, CSS, JS).


Untuk melihat catatan, pengunjung bisa mengakses materi-materi yang penulis buat di bagian kiri dan scroll untuk melihat ke bawah. Di bagian terdapat nama mata kuliah diikuti dengan daftar materi.


Terima kasih atas waktu dan perhatiannya untuk mengunjungi situs ini.
